# Source Acquisition

Source acquisition turns user-provided source inputs into crawl inventories.

Current sprint scope:

- URL list ingestion from CSV text.
- URL normalization and deduplication.
- Domain allowlist enforcement.
- Crawl inventory output using the W1-S1 source acquisition contracts.

Out of scope until later W1 sprints:

- Network fetches.
- Documentation portal discovery.
- JavaScript rendering.
- Canonical document generation.
