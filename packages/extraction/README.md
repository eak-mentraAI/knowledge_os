# Extraction

Extraction turns canonical documents and domain packs into evidence-linked node and edge candidates.

Current sprint scope:

- Deterministic Foundry seed-entity extraction.
- Evidence-linked node candidates.
- Evidence-linked `PROVIDES_CONTEXT_TO` edge candidate extraction.
- Extraction run metadata and metrics.

Out of scope until later sprints:

- LLM extraction.
- Prompt versioning.
- Entity resolution.
- Production confidence calibration.
