# Brand Design Shard

Knowledge OS is dark-first for MVP.

Brand attributes:

- Intelligent.
- Structured.
- Confident.
- Calm.
- Technical.
- Precise.
- Trustworthy.

Avoid:

- Playful.
- Gimmicky.
- Cyberpunk.
- Gaming-like.
- Generic enterprise portal.
- Graph database tooling.

Tokens:

```css
--bg-primary: #0B1020;
--bg-secondary: #12192B;
--bg-surface: #1B2336;
--brand-primary: #4DA3FF;
--brand-hover: #67B3FF;
--brand-dark: #2E7FE8;
--success: #2ECC71;
--warning: #F5A623;
--error: #FF5C5C;
```

Typography:

- Inter preferred.
- Geist acceptable.
- JetBrains Mono for technical identifiers.

UX:

- Hide graph complexity.
- Center understanding, decisions, dependencies, outcomes, project spaces, and assistants.
- Treat ChatGPT Projects, GPTs, SharePoint, Copilot, and similar tools as valid execution runtimes.
- Use architecture layers, dependency trees, coverage heatmaps, relationship tables, domain maps, and capability maps.
- Avoid force-directed graph visuals and animated graph clouds.

Logo:

- Use `docs/source/brand_assets/knowledge_os_logo.png`.
- Do not recreate or reinterpret it without a new approved brand asset.
