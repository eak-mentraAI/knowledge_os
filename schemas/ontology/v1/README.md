# Ontology Schemas v1

These schemas define the base ontology and future domain pack contract.

Current sprint scope:

- `ontology-schema.schema.json`
- `base-ontology.json`

Rules:

- The base ontology remains domain-agnostic.
- Domain packs may extend the base ontology.
- Domain packs may not redefine base node or edge types incompatibly.
- Security boundaries, governance controls, constraints, policies, and risks are first-class.
